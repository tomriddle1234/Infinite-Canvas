# Infinite-Canvas
Supports comfyui/API calls/modelscope calls
支持几乎所有API平台调用，修改API文件中的env的网址和key就可以运行。
支持本地comfyui调用，支持modelscope免费调用。

详细教程：https://youtu.be/1y9ShTvgC_w

设计了极为强大的分组功能，简化了页面显示和操作
